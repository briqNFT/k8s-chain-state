# k8s-chain-state

## Installation

```console
pip install k8s-chain-state
```

## Usage

```sh
# Create the custom resource definition
k8s-chain-state crd

# Add new contracts
k8s-chain-state contract starknet-testnet set-nft 0x05f9e1c4975b0f71f0b1af2b837166d321af1cdba5c30c09b0d4822b493f1347 
```
